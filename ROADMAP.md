# Roadmap

## M0: Spec v0.1 published
Acceptance criteria:
- Core model defined and documented.
- Psellos extension namespace defined.
- JSON Schema published for v0.1.
- Controlled vocabularies drafted.
- Example payloads provided.
- Version tag applied as v0.1.

## M1: Builder v0.1 produces compiled artifacts from demo dataset
Acceptance criteria:
- Builder pins to psellos-spec v0.1.
- Demo dataset from psellos-data is validated.
- Artifacts compiled deterministically.
- Artifact manifest includes spec version.
- Output format documented.
- Version tag applied as v0.1.

## M2: Web v0.1 loads artifacts and renders basics
Acceptance criteria:
- Web pins to builder outputs for spec v0.1.
- Artifacts load without errors.
- Basic graph visualization renders.
- Entity panel displays core fields.
- Layer toggle controls narrative layers.
- Version tag applied as v0.1.
